# Flask Commerce API

API RESTful de comércio eletrônico desenvolvida com **Python** e **Flask**.  
Gerencia produtos, autenticação de usuários, carrinho de compras e finalização de pedidos (checkout simulado).

## Funcionalidades Principais

- **Produtos**: Listagem, busca, criação, atualização e remoção (CRUD)
- **Autenticação**: Registro de usuários, login e proteção de rotas (provavelmente JWT ou sessions)
- **Carrinho**: Adição/remoção de itens, visualização do carrinho persistente
- **Checkout**: Finalização de compra com simulação de pagamento e criação de pedido
- Integração com banco de dados relacional (SQLite para dev, PostgreSQL/MySQL em prod)
